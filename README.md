# 📖 프로젝트 개요
안정적이고 효율적인 에너지 공급을 위해 **전력 사용량 예측의 중요성**은 점점 커지고 있습니다.
본 프로젝트는 100개 건물의 전력 데이터를 기반으로, 기상 데이터와 건물 특성을 활용하여 **건물별 전력소비량 예측 모델**을 개발하는 것을 목표로 했습니다.

- **배경**: 기후 변화, 에너지 전환 정책 → 수요 예측 기반 관리 필요
- **목표**: 산업 현장에 적용 가능한 예측 알고리즘 발굴
- **출력**: 7일간(2024.08.25~08.31) 건물별 전력 사용량 예측

---

## 📊 평가 지표 & 전략
- **SMAPE (Symmetric Mean Absolute Percentage Error)**
- **핵심 전략**:
  - Feature Engineering (기상학·공학적·시계열 피처)
  - XGBoost, LightGBM, CatBoost 모델링
  - 시드·폴드·모델별 앙상블
  - 건물별 개별 모델링 + 유형별 모델링

---

## 📂 데이터
- **train.csv**: 2024.06.01 ~ 08.24, 100개 건물의 기상·전력 데이터
- **building_info.csv**: 건물 특성 (유형, 연면적, 냉방면적, ESS, PCS 용량 등)
- **test.csv**: 2024.08.25 ~ 08.31 기상 데이터
- **sample_submission.csv**: 제출 양식

**EDA 주요 인사이트**
- 건물 유형별 소비 패턴 차이 존재 (예: IDC > 병원 > 학교 > 아파트 순)
- 일부 건물 이상치·불규칙 패턴 존재
- 여름철 냉방 수요에 따른 전반적 우상향 그래프

---

## 🛠 데이터 전처리 & 피처 엔지니어링
- **보간법**: CubicSpline 기반 향상된 보간 → 0값/이상치 처리
- **시간 인코딩**: sin/cos(hour, date, month, weekday)
- **기상 지수**: THI(온습도지수), CDH(냉방도), WCT(체감온도)
- **전력 통계 피처**: 요일-시간대별 평균/표준편차
- **건물별 휴무 패턴 반영**: 건물별 주말·특수 휴무일 처리

---

## 🧠 모델링
### 1️⃣ XGBoost
- 12-Fold 시계열 교차검증 + 시드 앙상블 (60개 모델)
- Weighted MSE (언더예측에 100배 가중치) 적용
- LB Score: 건물별 5.4208 / 유형별 5.7983

### 2️⃣ LightGBM
- 12-Fold 교차검증 + 폴드 앙상블 (12개 모델)
- 단계별 과소추정 보정
- LB Score: 건물별 5.3909 / 유형별 5.8168

### 3️⃣ CatBoost
- 로그 변환 + 2단계 앙상블
- LB Score: 건물별 5.4208 / 유형별 5.7983

---
